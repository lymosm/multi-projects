## Description ##

Xtoool Product Feed Plugin makes it easy to insert product feed ads on your WordPress pages and manage them in bulk.

Full documentation can be found at [https://www.xtoool.com/wordpress/product-feed/]

Are you still worried that the conversion rate of Woocommerce website is too low? This plugin provides a solution that allows you to quickly add and manage product display ad placements on your WordPress website pages to increase product exposure and increase sales.

# User's guidance #

1. Install Xtoool Product Feed Plugin
2. Add the products you want to display in "Add products"
3. Add advertising space in "Add products list"
4. Copy the shortcode of the ad slot to the target WordPress page
